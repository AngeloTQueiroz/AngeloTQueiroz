# 👩🏻‍💻 Angelo Teixerira

**`Desenvolvedor FullStack | Cloud`**

Me chamo Angelo Teixeira de Queiroz, sou de São Paulo. Trabalho com tecnologia desde jovem aprendiz, passando por análise de dados até hoje, atuando como estagiário de TI, onde lido com infraestrutura, Microsoft 365 e automação de processos. No caminho, fui me apaixonando por desenvolvimento de software, e hoje tenho experiência com React.js, Node.js, Java e bancos de dados como SQL Server, MySQL e PostgreSQL. Gosto de resolver problemas reais no dia a dia, seja automatizando uma rotina manual, seja construindo uma aplicação do zero, e estou sempre estudando algo novo — ultimamente com foco em cloud e segurança.
